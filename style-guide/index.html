Ideen bag design valg

Under projektet er der blevet undersøgt og kigget på forskellige webside design, med særligt fokus på sider der henvender sig til den ældre befolkning, samt andre højskoleforeningers sider. Her fandt teamet ud af at man ved design til ældre skal tage særligt hensyn til farve, kontrast og størrelser, for at de ældre kan bruge og navigere optimalt på siden.

Der er derudover blevet valgt en række farver, lavet en serie ikoner og illustrationer, inspireret af Odense Højskoleforenings egens side, samt den kendte Højskole sangbog, som de også selv benytter i mange af deres arrangementer.

Alle elementer på hjemmesiden er valgt med udgangspunkt i vores research, samt på baggrund af både Scneidermann og Shahaf's 8 design principper. I den forbindelse er der også blevet fundet nogel UI-Patterns som har inspireret de endelige design der er blevet lavet.

var i;

var dots = document.getElementsByClassName ("demo");
if (n > slides.length) {slideIndex = 1}

function openCity(evt, cityName) {
var i, tabcontent, tablinks;
tabcontent = document.getElementsByClassName("tabcontent");
for (i = 0; i < tabcontent.length; i++) {
tabcontent[i].style.display = "none";
}

tablinks = document.getElementsByClassName("tablinks");
for (i = 0; i < tablinks.length; i++) {
tablinks[i].className = tablinks[i].className.replace(" active", "");
}

document.getElementById(cityName).style.display = "block";
evt.currentTarget.className += " active";
}
